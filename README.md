# Samsung Website Frontend Clone

A frontend clone of the Samsung India website built using **HTML, CSS, and JavaScript**.  
This project recreates the landing page design with navigation menus, product sections, search functionality, cart page, login/register pages, and footer layout.

---

## 🚀 Features

- Samsung-style responsive navigation bar
- Hover-based dropdown menu
- Search overlay with predefined recommendations
- Product cards with Buy buttons
- Cart page layout
- Login and Register pages
- Footer section similar to Samsung website
- Smooth hover effects and basic animations
- Clean frontend structure using HTML, CSS, and JavaScript

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript
- Font Awesome / Icons
- VS Code

---

## 📌 Project Purpose

This project was created as a frontend practice project to understand and implement:

- HTML page structure
- CSS layouts using Flexbox and Grid
- Dropdown menus
- Search UI
- Product card design
- Page navigation
- Basic JavaScript DOM manipulation
- Website cloning and UI recreation
